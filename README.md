# 📊 Social Media Engagement Dashboard

## 🔍 Overview
This project is a Social Media Analytics Dashboard built using Power BI.  
It helps analyze user engagement like Likes, Comments, and Shares.

---

## 🎯 Objectives
- Track social media performance  
- Analyze engagement trends  
- Identify high-performing posts  

---

## 📊 Key Metrics
- Likes  
- Comments  
- Shares  
- Engagement Rate  

---

## 🛠️ Tools Used
- Power BI  
- Excel
